# Background Color Changer

A simple web application built using HTML, CSS, and JavaScript that allows users to change the webpage background color by entering a color name.

## Features

- Accepts user input for color names.
- Changes the webpage background dynamically.
- Displays an alert if no color is entered.
- Validates whether the entered color is supported by the browser.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How It Works

1. Enter a color name such as:
   - red
   - blue
   - green
   - pink
   - purple

2. Click the **Try** button.

3. The background color changes instantly.

4. If the input is empty or invalid, an alert message is shown.
